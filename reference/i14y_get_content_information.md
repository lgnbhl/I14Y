# Get the information of a nomenclature by identifier

Get the information of a nomenclature by identifier

## Usage

``` r
i14y_get_content_information(id = NULL)
```

## Arguments

- identifier:

  string. The identifier of the nomenclature.

## Value

a list

## Examples

``` r
i14y_get_content_information(
  identifier = "HCL_CH_ISCO_19_PROF"
)
#> Error in i14y_get_content_information(identifier = "HCL_CH_ISCO_19_PROF"): unused argument (identifier = "HCL_CH_ISCO_19_PROF")
```
